Commandes pour construire et tester
Pour construire l'image Docker :
```
docker build -t node-mapnik .
```
Pour exécuter le conteneur et vérifier que node-mapnik fonctionne :
```
docker run --rm node-mapnik
```

Cette commande affichera les informations de version de node-mapnik, les plugins disponibles et créera une image test.png avec un fond bleu.
Pour récupérer l'image générée sur votre machine hôte :
```
docker run --rm -v $(pwd):/output node-mapnik sh -c "cp /app/test.png /output/"
```

Si vous souhaitez accéder à un shell interactif dans le conteneur pour davantage de tests :
```
docker run --rm -it node-mapnik bash
```
